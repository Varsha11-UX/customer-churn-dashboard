# 📊 Customer Churn Prediction Dashboard

An end-to-end Machine Learning web application that analyzes customer churn patterns and predicts churn probability using a Random Forest classifier.

🔗 **[Live Demo](your-streamlit-link-here)**

---

## 🚀 Features

- **EDA Module** — Churn distribution, correlation heatmap, tenure & charges analysis
- **ML Model** — Random Forest with Accuracy, Precision, Recall, ROC-AUC metrics
- **Live Prediction** — Enter customer details and get real-time churn probability
- **Business Insights** — Feature importance + actionable retention recommendations

---

## 💡 Business Use Case

Customer churn costs businesses millions annually. This dashboard helps data and product teams:
- Identify high-risk customers early
- Understand key churn drivers (contract type, tenure, charges)
- Take proactive retention actions
